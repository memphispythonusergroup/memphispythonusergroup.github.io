## Foundry Toolkit for VS Code

The Foundry Toolkit for VS Code enables developers to build, test and deploy AI apps using models from OpenAI, Anthropic, Google, GitHub and more. It also works with local models using Ollama. In this session we will look at the features of the Foundry Toolkit for VS Code that help explore and evaluate models. We will also see how to build AI agents and deploy them to Microsoft Foundry.
